## GitHub Pages に公開する手順（初回）

### Step 1：GitHubにリポジトリを作成する

1. `https://github.com` を開いてサインイン
2. 右上の「＋」ボタン → 「New repository」をクリック
3. Repository name に `life-os` と入力
4. Public を選ぶ
5. 「Create repository」をクリック

### Step 2：ファイルをアップロードする

1. 作成したリポジトリのページで「uploading an existing file」をクリック
2. `life-os` フォルダの中身を全部ドラッグ＆ドロップ（フォルダごとではなく中身を選ぶ）
3. 「Commit changes」をクリック

### Step 3：Pages を設定する

1. リポジトリの「Settings」タブをクリック
2. 左側の「Pages」をクリック
3. Branch の欄で「main」を選ぶ
4. フォルダは「/ (root)」のまま
5. 「Save」をクリック
6. 数分待つと `https://（ユーザー名）.github.io/life-os/` でアクセスできる

## ファイルを更新して GitHub Pages に反映する手順

PowerShell を使う場合（Claude Code で作業後）：

```powershell
# まず life-os フォルダに移動
cd "c:\Users\az123\OneDrive\デスクトップ\claude code space\life-os"

# Git の設定（初回のみ）
git init
git remote add origin https://github.com/（ユーザー名）/life-os.git

# 変更をコミット・プッシュ
git add .
git commit -m "更新内容の説明"
git push origin main
```

GitHub のウェブ画面からアップロードする場合：
1. GitHub のリポジトリを開く
2. 更新したいファイルをクリック
3. 鉛筆アイコン（Edit）をクリック
4. 内容を貼り付けて「Commit changes」

## iPhone のキャッシュをクリアして最新版を見る

1. iPhone の Safari で `設定` → `Safari` を開く
2. 「履歴と Web サイトデータを消去」をタップ
3. 再度 `https://（ユーザー名）.github.io/life-os/` を開く
4. ホーム画面に追加し直す（古いアイコンは削除してから追加）

> **注意**: キャッシュを消すと localStorage のデータも消えることがある。
> データが消えた場合は Claude Code にバックアップ機能の追加を依頼してください。

## Claude Code に追加作業を依頼するときの伝え方

作業を依頼するときは以下を伝えると早く対応できます：

1. どのページ（どの機能）を変えたいか
2. 何を追加・変更・削除したいか
3. 今どんな問題が起きているか（エラーメッセージがあれば貼る）

例：「リマインダーに『時刻』の入力欄を追加してほしい」
例：「在庫管理でカテゴリが五十音順に並ぶようにしてほしい」
