# 生活OS

自分専用の生活管理 Web アプリです。

## 機能一覧

| アイコン | 機能 | ファイル |
|---------|------|---------|
| ⏰ | リマインダー | pages/reminder.html |
| 🔌 | 家電メモ | pages/appliance.html |
| 🧹 | 掃除手順 | pages/cleaning.html |
| 👕 | 洗濯手順 | pages/laundry.html |

## 技術仕様

- HTML / CSS / JavaScript のみ
- データは localStorage に保存
- GitHub Pages で公開可能
- iPhone ホーム画面に追加して PWA 的に使用
