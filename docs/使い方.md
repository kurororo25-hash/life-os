# 使い方ガイド - 生活OS

---

## ブラウザで開く方法（PC）

1. エクスプローラーで `life-os` フォルダを開く
2. `index.html` をダブルクリックする
3. ブラウザで開いたらホーム画面が表示される

---

## GitHub Pages で公開する手順

1. GitHubにリポジトリを作成する（例：`life-os`）
2. `life-os` フォルダの中身を全部アップロードする
3. リポジトリの「Settings」→「Pages」→「Branch: main」→「/ (root)」を選んで保存
4. 数分後に `https://ユーザー名.github.io/life-os/` でアクセスできるようになる

---

## iPhoneのホーム画面に追加する手順

1. iPhoneのSafariで `https://ユーザー名.github.io/life-os/` を開く
2. 画面下の共有ボタン（四角から矢印が出るアイコン）をタップ
3. 「ホーム画面に追加」をタップ
4. 名前を「生活OS」のまま「追加」をタップ
5. ホーム画面にアイコンが追加される

> ホーム画面から開くとアドレスバーなしで全画面表示になる

---

## 各機能の使い方

### ⏰ リマインダー

- 右下の ＋ ボタンでリマインダーを追加
- タイトル・日付・優先度・繰り返しを設定する
- リストの丸いボタンを押すと完了になる
- 繰り返し設定済みは、完了を押すと次の日付に更新される

### 📦 在庫管理

- ＋ ボタンで品名・カテゴリ・現在数・最低在庫数を登録
- − / ＋ ボタンで在庫数を増減できる
- 最低在庫数以下になると「在庫少」バッジが出る
- 「買う」ボタンで買い物リストに追加できる
- 上部の「まとめて買い物リストへ」ボタンで在庫少を一括追加できる

### 🛒 買い物リスト

- 在庫管理から自動追加 or 下部フォームで手動追加
- 丸ボタンを押すと購入済みになる
- 在庫から来た項目は購入済みにすると在庫数が +1 される
- ヘッダーの ✅ ボタンで購入済みをまとめて削除できる

### 🔌 家電メモ

- ＋ ボタンで名前・メーカー・型番・購入日・保証期限を登録
- 保証が3ヶ月以内に切れると「保証まもなく切れ」が表示される

### 🧹 掃除手順 / 👕 洗濯手順

- ＋ ボタンでタイトルと手順を登録（1行につき1ステップ）
- リストをタップすると手順が番号付きで表示される

### 🚐 車内荷物

- ＋ ボタンで荷物を登録
- 置き場所ごとにグループ表示される

### 💼 仕事メモ / 📚 学習メモ / ❤️ 健康メモ / 💪 トレーニング

- ＋ ボタンでタイトル・内容・タグを登録
- 📌 ボタンでピン留め（上部に固定）できる
- タップで全文表示

### 📓 日記・ログ

- ＋ ボタンで日付・気分・内容を記録
- 気分は 😄🙂😐😔 から選ぶ
- 年月でグループ表示される

## データの保存について

- データはすべてブラウザの **localStorage** に保存される
- ブラウザのキャッシュ削除・プライベートモードではデータが消えるので注意
- iPhoneとPCのデータは別々（同期しない）
