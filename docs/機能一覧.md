# 機能一覧

---

## ⏰ リマインダー

| 操作 | 説明 |
|------|------|
| 追加 | ＋ ボタンから追加 |
| 完了 | 丸いチェックボタンをタップ |
| 繰り返し | 毎日・毎週・毎月から選択。完了を押すと次の日付に自動更新 |
| 優先度 | 高（赤）/ 中（黄）/ 低（緑）のバッジで表示 |
| タブ | 未完了 / 今日 / 完了済み で切り替え |
| 期限切れ | 日付が過ぎた未完了は「期限切れ」バッジ付きで赤背景 |

---

## 📦 在庫管理

| 操作 | 説明 |
|------|------|
| 追加 | 品名・カテゴリ・単位・現在数・最低在庫数・保管場所を登録 |
| 数量変更 | − / ＋ ボタンで1つずつ増減 |
| 在庫少アラート | 現在数 ≤ 最低在庫数 のとき警告バッジ＋黄背景 |
| 買い物リスト追加 | 「買う」ボタンで個別追加、バナーの「まとめて」で全一括追加 |
| 表示 | カテゴリ別グループ表示 |

---

## 🛒 買い物リスト

| 操作 | 説明 |
|------|------|
| 追加 | 下部フォームから手動追加（品名・数量・カテゴリ） |
| 在庫連携 | 在庫管理から自動追加された項目は「在庫連携」タグ付き |
| 購入済み | 丸ボタンをタップ。在庫連携の場合は在庫数が +1 される |
| 削除 | 個別は 🗑️ ボタン。ヘッダー ✅ で購入済みを一括削除 |

---

## 🔌 家電メモ

| 登録項目 | 説明 |
|---------|------|
| 名前 | 必須 |
| メーカー・型番 | 任意 |
| 設置場所 | 任意 |
| 購入年月 | 年月で記録（YYYY-MM） |
| 保証期限 | 3ヶ月以内に切れる → 「保証まもなく切れ」バッジ |
| 購入価格・メモ | 任意 |

---

## 🧹 掃除手順 / 👕 洗濯手順

| 操作 | 説明 |
|------|------|
| 追加 | タイトル・頻度・所要時間・手順（1行1ステップ）を登録 |
| 閲覧 | タップで手順を番号付きで表示 |

---

## 🚐 車内荷物管理

- 車種情報（エブリィ・車高 1.8m・荷室長 190cm）を上部に常時表示
- 置き場所でグループ表示（助手席足元・荷室など）
- 「常時積み」/「必要時のみ」で区別

---

## 🔧 整備管理

| 操作 | 説明 |
|------|------|
| 追加 | プリセット（エンジンオイル・オイルエレメント・車検・ブレーキ・タイヤ・バッテリー等）または自由入力で登録 |
| 登録項目 | 前回実施日・頻度（月数）・目安走行距離(km)・前回実施時の走行距離(km)・メモ |
| 走行距離 | 上部で現在の走行距離を手入力（km基準の項目の残りkm計算に使用） |
| 並び順 | 日数・距離のうち早く来る基準で緊急度を計算し、対応が必要な順（期限切れ→まもなく→余裕あり）に自動ソート |
| 実施済み | 「✅ 実施済み」ボタンで前回実施日・走行距離を今日/現在値にリセットし次回サイクルへ |

---

## 💼 仕事メモ / 📚 学習メモ / ❤️ 健康メモ / 💪 トレーニングメモ

共通機能（`memo-base.js` で実装）：

| 操作 | 説明 |
|------|------|
| 追加 | タイトル・内容・タグ（スペース区切り）を登録 |
| ピン留め | 📌 ボタンで上部に固定 |
| 検索 | タイトル・内容・タグを横断検索 |
| 閲覧 | タップで全文表示 |

---

## 📓 日記・ログ

| 操作 | 説明 |
|------|------|
| 追加 | 日付・気分（😄🙂😐😔）・内容を記録 |
| 表示 | 年月別グループ、新しい順 |

---

## ホーム画面のバッジ

| バッジ | 表示条件 |
|-------|---------|
| リマインダー | 今日以前の未完了件数 |
| 在庫管理 | 最低在庫数以下の品目数 |
| 買い物リスト | 未購入件数 |
| 整備管理 | 期限切れ・まもなく（残り15%以内）の項目数 |
