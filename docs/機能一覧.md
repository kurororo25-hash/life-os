# 機能一覧

---

## ⏱️ タイムボクシング

| 操作 | 説明 |
|------|------|
| 表示切り替え | 「1日」タブ＝縦のタイムライン表示（朝2:00〜夜21:00）／「1週間」タブ＝日〜土の7日間をスクロール不要で一望できるグリッド表示 |
| 追加 | タイムライン（1日／1週間どちらも）の空いている場所をタップ、または ＋ ボタンから追加 |
| 入力項目 | 「やること」（テキスト）と「開始時刻」（30分刻みのセレクト）のみ。所要時間は30分固定なので終了時刻の入力は不要 |
| 日付移動 | ヘッダーの ‹ › ボタン（1週間表示では週単位で移動）または日付タップ（カレンダー）で移動。「今日」ボタンで即戻れる |
| 重なり表示 | 同じ時間帯に予定が複数ある場合はタイムライン上で横に並べて表示 |
| 完了 | リストの丸ボタン、またはタイムラインのブロックをタップ→編集モーダルのスイッチで完了にできる（完了はグレーアウト＋取り消し線） |
| 訂正 | リストの ✏️ で編集、🗑️ で削除。タイムライン上のブロックをタップしても編集モーダルが開く |
| サマリー | 表示中の期間の件数・合計時間・完了数をヘッダー下に表示 |
| Googleカレンダー連携 | ヘッダーの📅ボタンからログイン。保存時に自動でGoogleカレンダーにも追加。「未完了を全て同期」でまとめて反映も可能（リマインダーと同じ仕組み） |

## 📔 ライフログ

| 操作 | 説明 |
|------|------|
| 用途 | その日一日を振り返るための記録（タイムボクシングと同じ縦タイムラインUIだが、こちらは実際にやったことを過去形で記録する） |
| 追加 | タイムラインの空いている場所をタップ、または ＋ ボタンから追加 |
| 入力項目 | 「やったこと」・「開始〜終了時刻」（30分刻み、自由な長さ）・「気分」（😄🙂😐😔、任意）・「メモ」（任意） |
| 連続入力 | 保存すると次の記録の開始時刻が自動で「今回の終了時刻」にセットされ、モーダルを開いたまま次を入力できる（タイムボクシングの続けて追加と同じ仕組み） |
| 日付移動 | ヘッダーの ‹ › ボタンまたは日付タップ（カレンダー）で移動。「今日」ボタンで即戻れる |
| 重なり表示 | 同じ時間帯に記録が複数ある場合はタイムライン上で横に並べて表示 |
| 訂正 | リストの ✏️ で編集、🗑️ で削除。タイムライン上のブロックをタップしても編集モーダルが開く |
| サマリー | その日の件数・合計記録時間をヘッダー下に表示 |
| リマインダー/カレンダー連携 | なし（振り返り専用のため） |

## ⏰ リマインダー

| 操作 | 説明 |
|------|------|
| 追加 | ＋ ボタンから追加 |
| 完了 | 丸いチェックボタンをタップ |
| 繰り返し | 毎日・毎週・毎月から選択。完了を押すと次の日付に自動更新 |
| 優先度 | 高（赤）/ 中（黄）/ 低（緑）のバッジで表示 |
| タブ | 未完了 / 今日 / 完了済み で切り替え |
| 期限切れ | 日付が過ぎた未完了は「期限切れ」バッジ付きで赤背景 |

## 🔌 家電メモ

| 登録項目 | 説明 |
|---------|------|
| 名前 | 必須 |
| メーカー・型番 | 任意 |
| 設置場所 | 任意 |
| 購入年月 | 年月で記録（YYYY-MM） |
| 保証期限 | 3ヶ月以内に切れる → 「保証まもなく切れ」バッジ |
| 購入価格・メモ | 任意 |

## 🧹 掃除手順 / 👕 洗濯手順

| 操作 | 説明 |
|------|------|
| 追加 | タイトル・頻度・所要時間・手順（1行1ステップ）を登録 |
| 閲覧 | タップで手順を番号付きで表示 |

## 🔧 整備管理

| 操作 | 説明 |
|------|------|
| 追加 | プリセット（エンジンオイル・オイルエレメント・車検・ブレーキ・タイヤ・バッテリー等）または自由入力で登録 |
| 登録項目 | 前回実施日（任意）・次回予定日・次回予定の走行距離(km)・メモ。次回予定日と走行距離はどちらか一方または両方を手動入力 |
| 走行距離 | 上部で現在の走行距離を手入力（次回予定の走行距離との差分計算に使用） |
| 日付入力 | 年→月→日を個別のセレクトで入力。年を選ぶと自動で月へ、月を選ぶと自動で日へフォーカスが移動 |
| 並び順 | 期限切れ→まもなく→余裕ありの順にソートし、同じ区分内では次回予定日・距離が近い順に並ぶ |
| 編集・削除 | ✏️ で次回予定日・距離を手動で更新、🗑️ で削除 |

|-------|---------|
| リマインダー | 今日以前の未完了件数 |
| 整備管理 | 期限切れ・まもなく（残り14日 or 300km以内）の項目数 |
